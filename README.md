# Al Quran App

Build with TailwindCSS and Next.js

### Install dependencies:

```
$ npm i
or
$ yarn 
```

### Development

To start the development server run:

```
$ npm run dev
or
$ yarn dev
```


### Production

To build the application bundle run:

```
$ npm run build
$ npm run start
or
$ yarn build
$ yarn start

```
